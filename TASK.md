# Build 5 Upwork Demo Pages

Create 5 standalone HTML demo pages for Upwork proposals. Each should be a single HTML file with embedded CSS/JS. Modern, dark theme, professional, interactive. NO external dependencies.

## Demo 1: `demo-discord-server-setup.html`
**Job: Discord & Whop Server Setup for Sales Training Program ($250)**
Show an interactive Discord server structure mockup:
- Visual tree of categories & channels (General, Training, Announcements, Support, Community)
- Role hierarchy visualization (Admin > Coach > Moderator > Member)
- Permission matrix showing what each role can access
- Bot integrations listed (MEE6, Carl-bot, Ticket Bot)
- Whop subscription integration flow
- Make it look like a professional server blueprint

## Demo 2: `demo-openclaw-twitter-agent.html`
**Job: OpenClaw AI Agent for X/Twitter Outreach Campaign ($150)**
Show an OpenClaw agent workflow:
- Architecture diagram: OpenClaw → Skill → X/Twitter API → Approval Flow (Telegram)
- Live-looking campaign tracker dashboard (mock data: pitches sent, replies, rejections)
- Sample personalized pitch preview
- Daily progress tweet auto-generator
- Notion/Sheet tracker mockup with columns: Date, Target, Pitch, Status, Reply

## Demo 3: `demo-ai-community-manager.html`
**Job: AI Community Manager for Arts & Tech Community ($750)**
Show an AI-powered community management dashboard:
- Multi-platform view (Discord + Telegram side by side)
- Content calendar with auto-generated posts
- Member engagement metrics (active users, messages/day, growth chart)
- AI moderation log (spam detected, auto-responses)
- Event scheduling interface
- Welcome flow automation visualization

## Demo 4: `demo-api-automation.html`
**Job: PAX8 API Automation ($80)**
Show a clean API automation dashboard:
- Live-looking terminal with API calls being executed
- Bulk operation progress (Creating customers... 45/100)
- Operation log with color-coded status (success/error/pending)
- Stats: Customers Created, Licenses Provisioned, Errors Handled
- Code snippet preview showing clean Python/Node automation
- Error handling & retry visualization

## Demo 5: `demo-discord-sports-betting.html`
**Job: Discord Community Manager for Sports Betting ($200)**
Show a sports betting Discord server mockup:
- Server structure with betting-specific channels (Daily Picks, Live Games, Results, Education)
- Whop subscription tier visualization (Free, Pro $29/mo, VIP $99/mo)
- Embed preview of a "Daily Pick" post with odds, analysis, confidence rating
- Moderation bot setup (anti-spam, verification)
- Member growth projection chart
- Subscription revenue calculator

## Design Requirements
- Dark theme (#0d1117 background, similar to Discord/GitHub dark)
- Accent colors: Use a unique accent per demo (blue, green, purple, orange, teal)
- Smooth CSS animations (fade-in, slide-up on scroll)
- Responsive layout
- Professional typography (system fonts)
- Each page should have a header with "Built by Janis — Web Developer & Automation Specialist"
- Footer with "Demo for [Client Name] — Custom solution preview"
- Interactive elements: clickable tabs, expandable sections, animated counters
